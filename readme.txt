#Anti-Fake-Product-System using Blockchain

A simple decentralized Inventory System, using the Ethereum Blockchain, Solidity, MongoDB, 
Express.js, ReactJS, and Node.js stack...!



#Deploying Smart Contract

1. Installin Ganache and Create a Workspace with port 8545.

2.
npm install -g truffle.

Deploying Smart Contract to Blockchain:

3. 
cd blockchain

4. 
truffle migrate --reset

5. 
localhost 8545 in MetaMask


#Running the react development server

1.
cd blockchain

2. 
npm install

2.
set NODE_OPTIONS=--openssl-legacy-provider

3. 
npm start


#Running the node server

1.  
cd server

2.  
npm install

3.  
npm run dev
